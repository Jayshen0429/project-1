APP 智能水族箱購物車



隨著現代生活節奏的加快，人們對於擁有放鬆身心的空間日益渴望。
水族箱作為一種獨特的生態空間，不僅能夠美化居家環境，更能帶來心靈寧靜。
然而，傳統的水族箱管理常常需要耗費大量的時間和精力。
我們開始思考如何透過技術的力量，創造一個更智能、更便捷的水族箱生態體驗。

詳細資訊

About Project

Android Studio

使用程式語言Kotlin作為開發，設計性強調直覺，提升使用者體驗。

Code

使用 Fragment 可以確保介面的統一性，並方便在不同設備間進行適配。

HttpRequest

發送一個 HTTP 請求到Respberry。
控制Respberry上的硬體設備（如餵食器、水泵、LED 等）。

SQL & Firebase

通過MySQL儲存資料，使用者可以查看目前水族箱溫度及PH質。
商家可以購過 Firebase 查看消費者下訂資料。

Webcam

Mjpg-streamer 是用來將視訊網路串流化的工具，透過其內建的 MJPG 解碼器，可以提供良好的影像擷取速率和畫質表現。

Cart

使用 RecyclerView 將10幾種商品呈現，能提升使用者的流暢體驗。

APP Features

即時影像處理技術讓您無論身在何處，都能觀察每一條魚的動態和每一株水草的變化，彷彿身臨其境。

即時顯示水族箱的溫度和 pH 值，不論是白天還是夜晚，您都能即時了解水質狀況，為您的水生朋友提供最貼心的照顧。

除了水中飼養，透過購物您還可以探索更多滿足，讓您的水族箱擁有更多變化!

提供超過 10 種水族箱裝置布置方案，讓您的水族箱更具個性化，打造理想中的水下世界。

點選按鈕可以控制LED燈、餵食器、抽水，享受與寵物互動的樂趣!

消費者下單後，店家可以透過 Firebase 查看相關的訂單資訊，以便有效地處理訂單並提供優質的服務。
